## [0.1.0] - 2026-03-06

### Added

- Initial public release of `@adech/themes`.
- Published the `Superior` theme as the first available theme in the Adech collection.
- Added JSON token export for `Superior`.
- Added CSS custom properties export for `Superior`.
- Added Tailwind CSS v4 export for `Superior`.
- Added Tailwind CSS v3 preset export for `Superior`.
- Added public subpath exports for:
  - `@adech/themes/superior`
  - `@adech/themes/superior/json`
  - `@adech/themes/superior/css`
  - `@adech/themes/superior/tailwind`
  - `@adech/themes/superior/preset`

### Notes

- Current token naming is palette-based, using the pattern `adech-{palette}-{step}`.
- `Superior` currently includes palettes such as `boulevard`, `venomous`, and `swamp`.
